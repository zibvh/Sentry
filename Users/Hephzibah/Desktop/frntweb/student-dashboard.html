HTML Tables

HTML tables are used to display data in a grid of rows and columns. They're perfect for structured information like schedules, pricing tables, and comparison charts.

Basic Table Structure

Every HTML table uses the <table> element as the wrapper. Inside, you define rows with <tr>, header cells with <th>, and data cells with <td>.

<!-- A simple HTML table --> <table> <thead> <tr> <th>Name</th> <th>Age</th> <th>City</th> </tr> </thead> <tbody> <tr> <td>Fatima</td> <td>22</td> <td>Kano</td> </tr> </tbody> </table>

Table Sections

Good practice is to divide your table into three sections: <thead> for the header row, <tbody> for the main data, and <tfoot> for summary rows. This improves readability and helps with styling.

📊   [Table diagram image goes here]

Merging Cells

Use colspan to make a cell span multiple columns, and rowspan to span multiple rows. This is useful for creating complex table layouts.
